
# EcoTrack: Empowering citizens and cities through data-driven sustainability.

EcoTrack is a one-stop sustainability platform designed to bridge the gap between individual action and civic data. By centralizing carbon tracking, resource monitoring, and community engagement, EcoTrack transforms fragmented environmental data into a roadmap for a greener futureâ€”all without the use of AI/ML.

Target Users: Environmentally conscious individuals, NGOs, and local government bodies.

Core Value: Gamifying sustainability through rule-based logic and real-time data visualization to reduce carbon footprints and optimize city-level resource management.

Tech Stack
Frontend: React.js + Vite (Tailwind CSS for styling)

Backend: Node.js + Express.js

Database: MySQL (Structured relational data for users, habits, and events)

APIs: OpenWeather AQI API

Installation 

1)Clone the Repository

Bash

git clone https://github.com/yourusername/ecotrack.git
cd ecotrack


2)Frontend Setup

Bash

cd green-heartbeat-main
npm install
npm run dev
